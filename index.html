CSS свойство background. Фон для сайта.

Background управляет фоном элемента.

Это универсальное свойство CSS позволяет одновременно задать несколько параметров фона, а именно определить цвет, установить фоновое изображение, задать начальное положение фонового изображения, определить как будет повторяться фоновое изображение, установить будет ли прокручиваться фоновое изображение с содержимым и масштабировать фоновое изображение согласно заданным размерам. Так же значением background может быть указана градиентная заливка фона.

  1. background-color
  2. background-image
  3. background-repeat
  4. background-position
  5. background-attachment
  6. background-size
  7. Установление нескольких изображений для блока

Определяет как будет повторяться фоновое изображение.

По умолчанию.

repeat-y

background-position

Задает начальное положение фонового изображения, где первое значение (px, проценты, слово) определяет положение по горизонтали, второе - по вертикали.
